# Keybindings

| Type			| Command 	|
|:--------------|:----------|
| HyperLink		| ⌘L      	|
| Image			| ⇧⌘L      	|
| Bold      	| ⌘B       |
| Italic    	| ⌘I       |
| Strikethrough | ⇧⌘X	|
| Code    		| ⌘K		|
| Quote    		| ⌘U		|
| Bulleted List | ⇧⌘O		|
| Numbered List | ⇧⌘N		|
| Checked List 	| ⇧⌘Y		|
| Table        	| ⌘T		|
| New Post      | ⌘N		|
