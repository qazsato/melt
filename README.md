# Keybindings

| Type          | Command |
| :------------ | :------ |
| HyperLink     | ⌘L      |
| Image         | ⇧⌘L     |
| Bold          | ⌘B      |
| Italic        | ⌘I      |
| Strikethrough | ⌘D      |
| Code          | ⌘K      |
| Quote         | ⌘U      |
| Bullet List   | ⇧⌘U     |
| Ordered List  | ⇧⌘O     |
| Task List     | ⇧⌘X     |
| Table         | ⌘T      |
